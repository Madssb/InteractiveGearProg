Chart Generating Example

Making items appear

  • data/sequence.json Decides what items are shown in the chart, and their sequencing. Example: [[a],[b,c],[d]] a renders as first item, b and c as a group in the middle, and d as the last item.
  • data/generated/items.json provides the required image- and wikiUrls per item that the chart requires, and can be generated with a provided python script
  • tools/build_items_json.py is the aforementioned python script which generates the items.json needed based on the items it finds in sequence.json. It requires the osrswiki_images python package.
